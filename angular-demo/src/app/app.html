🧠 Smart Variable Naming Test

Use the Smart Text to i18n plugin on these template literals to see semantic variable naming in action!

📋 How to Test in Angular:
  1. 1. Open src/app/app.ts in IntelliJ IDEA
  2. 2. Select a template literal (lines 86, 97, or 109)
  3. 3. Right-click → "Smart Text to i18n" → Extract
  4. 4. Verify it generates semantic placeholders instead of {{ '{0}' }}, {{ '{1}' }}
  5. 5. Check it uses this.translate.get() not t()
